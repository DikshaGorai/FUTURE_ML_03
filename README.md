Task 3: Resume / Candidate Screening System - 
​The Resume Screening System is a Machine Learning-driven solution designed to automate the initial stages of recruitment by screening and ranking candidate resumes against specific job descriptions. This project leverages Python and advanced text analysis tools (spaCy/NLTK) to perform resume parsing, text cleaning, and sophisticated feature extraction. By utilizing Scikit-learn to build ranking models, the system evaluates candidates based on their role fit, extracts key skills, and identifies potential skill gaps relative to the job requirements. The final deliverable is an intelligent screening tool that provides recruiters with a ranked list of candidates and highlights the specific strengths and missing qualifications of each applicant. This task showcases expertise in text analysis, resume scoring, and the practical application of ranking models in HR technology.
